## ShortLinkApp

# Mac files

1.  **Запуск через терминал**

    ```shell
    dotnet run
    ```

2.  **Перейти  по ссылке**

    ```shell
    http://localhost:5056/index.html
    ```

3.  **Авторизоваться**

    ```shell
    Можно создать свой акк, или войти уже в зарегистрированный
    login: qwe
    password: 123qwe
    ```
4.  **Выключение через терминал**

    ```shell
    Сочетание клавишь: cntrl + C
    ```
